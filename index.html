How it works

From fresh install
to coding in minutes.

Nuvio handles everything ‚ÄĒ package manager detection, service configuration, and cleanup ‚ÄĒ so you don't have to.

01

Run the bootstrap

The curl | bash command downloads and runs bootstrap.sh, which clones the full repo to ~/.nuvio.

02

Answer the prompts

Enter your Linux username, projects folder name, and MySQL root password. Hit enter to accept the defaults.

03

Confirm and sit back

Review the summary and confirm. Nuvio installs and configures everything with clean progress output.

04

Start building

Use sudo new-site myapp to scaffold your first local project at myapp.test instantly.
